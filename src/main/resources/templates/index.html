AWS Asynchronous Photo Analyzer Application

The AWS Photo Analyzer application is a sample application that uses the Amazon Rekognition Service as well as other AWS Services and the Java V2 SDK. This example application uses the Asynchronous client for the Amazon S3 and Amazon Rekognition Services. Analyzing nature photographs has never been easier! Simply perform these steps:

  1. Upload a nature photograph to an Amazon S3 bucket by choosing the Upload Photos menu item.
  2. Choose Choose File and browse to a nature image located on your desktop.
  3. Choose Upload to upload your image to an S3 bucket.
  4. Choose Get Images to view the images located in the S3 bucket. All images in the bucket are displayed in the table.
  5. Analyze the photographs and produce a report by choosing the Analyze Photos menu item.
  6. Enter an email address in the email field and choose Analyze Photos.
  7. Amazon SES is used to send an email with an Excel report to the specified email recipient.
